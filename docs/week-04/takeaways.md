## 本周核心认知清单

- [ ] GPU 的本质：大量简单计算单元并行处理相同指令的工厂，不是"更强的 CPU"
- [ ] SM（Streaming Multiprocessor）是 GPU 的基本工作单元——H100 有 132 个 SM
- [ ] CUDA Core 是通用计算单元，Tensor Core 是矩阵乘法专用单元（速度 4-16 倍）
- [ ] Warp（32 线程一组）是 GPU 最小执行单位，SIMT 模式——一个指令 32 个线程同步执行
- [ ] 内存层次：寄存器→共享内存→L2 缓存→HBM，越近越快越小
- [ ] HBM 是 GPU 的命门——仅 SK Hynix 和 Samsung 两家能量产，双寡头垄断
- [ ] Memory Wall（内存墙）：计算速度增长远快于内存带宽增长，内存成为真正瓶颈
- [ ] Blackwell 三大设计决策：双芯封装、FP4 精度、NVLink 5.0
- [ ] 理论算力 vs 实际算力：MFU 通常只有 30-50%，三大损耗——内存瓶颈/通信开销/流水线气泡
- [ ] NVIDIA 护城河结论：硬件领先 2-4 年可追赶，CUDA 生态领先 10+ 年难替代
- [ ] 飞轮效应：硬件领先→更多开发者→更丰富生态→更多企业选购→更多研发投入→更强硬件
- [ ] NVIDIA GPU 毛利率 75-85%，定价权来自系统级切换成本而非硬件独占

## 关键数字速查

| 数字 | 含义 |
|------|------|
| 132 个 SM | H100 的流式多处理器数量 |
| 16,896 | H100 的 CUDA Core 数量 |
| 528 | H100 的 Tensor Core 数量 |
| 32 | 一个 Warp 包含的线程数（GPU 最小执行粒度） |
| 3.35 TB/s | H100 HBM3 显存带宽 |
| 8.0 TB/s | B200 HBM3e 显存带宽 |
| 2,080 亿 | B200 双芯封装的晶体管总数 |
| 858 mm² | 光罩极限（单芯片最大面积） |
| 40-50% | 典型训练任务的 MFU 范围 |
| 75-85% | NVIDIA GPU 产品毛利率 |
| ~80% | NVIDIA 在 AI 训练 GPU 市场的份额 |
