## 本周核心认知清单

- [x] 数据中心的本质：能量转换与热管理系统，不是"放服务器的房子"
- [x] 风冷物理极限在 25-30kW/柜，AI 训练机柜 40-120kW 必须液冷
- [x] 冷板液冷（Cold Plate）是当前主流路线：B200 原生支持、供应链成熟、运维友好
- [x] 选址五大核心变量：电力、网络、气候、土地、政策
- [x] 模块化预制（Modular Prefab）数据中心 6-9 个月可交付，是 AI 算力抢跑的关键手段
- [x] 决策优先级方法论："先排除硬约束，再优化软变量"
- [x] GPU 出口管制（Export Control）是中国企业出海建 DC 的最核心驱动力，不是电价
- [x] 双轨策略（Dual-Track Strategy）：海外训练（获取高端 GPU）+ 国内推理（数据安全合规）
- [x] 混合策略（Hybrid Strategy）：现有风冷 DC 改造用于推理，新建液冷 DC 用于训练，不浪费已有资产
- [x] 内蒙古 vs 马来西亚的判断修正：推理选内蒙古（电价+安全），训练选海外（GPU 性能差距 3-5x 压过电价优势）

## 形成的判断框架

### 框架 1：先排除硬约束，再优化软变量
面对多变量决策时，先识别不可绕过的物理/技术/法规硬约束（如风冷极限、出口管制），排除不可行选项后，再在可行域内优化成本、时间、运营等软变量。避免在不可行方案上浪费分析资源。

### 框架 2：混合策略——已有资产再利用
改造 vs 新建不是二选一。已有资产如果在降级场景（如推理）中仍可用，就不应该直接废弃。正确做法是按场景分流：高需求场景新建，低需求场景改造复用。

### 框架 3：双轨策略——海外训练 + 国内推理
在出口管制背景下，中国 AI 企业的最优部署策略是地理分离：训练集群放海外（合法获取 H100/B200），推理服务放国内（数据安全、低时延、政策合规）。GPU 性能差距（3-5x）远大于电价差异，性能约束是第一优先级。
