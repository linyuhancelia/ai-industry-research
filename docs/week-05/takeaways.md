## 本周核心认知清单

- [ ] Google TPU 不是"更好的 GPU"，而是从头设计的 AI 专用芯片（ASIC）
- [ ] TPU 的核心优势是垂直整合：芯片+互联+网络+编译器+框架+模型全部自研
- [ ] Google 做 TPU 的根本动机是供应链自主权 + 长期成本优势，不是"做更好的芯片"
- [ ] AMD MI300X 硬件参数超过 H100，但实际训练 MFU 低 10-20 个百分点（ROCm 差距）
- [ ] AMD 的最优突破口是推理市场——HBM 大（192GB）+ 价格低 + CUDA 依赖弱
- [ ] 华为昇腾在中国市场的竞争对手是 H20（阉割版），不是 H100——这个对比下华为有竞争力
- [ ] 出口管制是华为昇腾市场需求的结构性驱动力
- [ ] ASIC（专用芯片）在特定场景性能极致，但缺乏生态和通用性
- [ ] GPU vs ASIC 终局：训练市场 GPU 主导，推理市场 GPU+ASIC 共存，边缘推理 ASIC 主导
- [ ] 推理市场竞争远比训练激烈——CUDA 飞轮在推理场景转得更慢
